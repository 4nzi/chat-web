## 仕様

- 1:1でチャットできる
- 友達一覧が見れる
    - 未読メッセージのカウント
    - 最新メッセージのプレビュー
    - メッセージが最新の友達を昇順に表示

## 技術

### フロントエンド

- React(Next.js)
    - SSR x SPA
- 状態管理(候補)
    - Redux Tool Kit
    - React Hooks
    - Jotai
- 実行環境(候補)
    - Vercel
    - Cloud Functions

### バックエンド
- Firebase

    - Authentication

    - Storage
