# chat-web

ブラウザ上で1対1のチャットが行えるサービスです。

### テストユーザー
Email: `test@example.com`  
Password: `testpass`

# 機能

- 1:1でのリアルタイムチャット
- 友達一覧
- IDでの友達追加
- 最新メッセージのプレビュー
- メッセージが最新の友達から昇順に表示
